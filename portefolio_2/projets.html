Projet 1 : Mini Game

Conception et développement d'un mini-jeu vidéo terminal en Python, intégrant un personnage, des objets, des niveaux générés aléatoirement, et des fonctionnalités avancées telles que la destruction de murs. Cette réalisation constitue une démonstration concrète de compétences en programmation de jeux, illustrant une maîtrise des concepts de manipulation de données et d'interaction utilisateur.

Langage utilisé : Python

Télécharger le projet

Résumé :

Ce projet, réalisé dans le cadre du TD MIASHS Informatique S3, vise à concevoir un mini-jeu vidéo jouable dans un terminal en utilisant le langage de programmation Python. Les principales fonctionnalités du jeu comprennent la création et l'affichage d'une carte, la représentation d'un personnage, la possibilité de déplacement, la génération aléatoire de niveaux, la collecte d'objets, et d'autres améliorations.

Fonctionnalités clés :

  • Affichage de base : Une fonction display_map qui affiche une carte représentée par une matrice et un dictionnaire associant des caractères aux nombres.
  • Personnage : Une fonction create_perso qui crée un personnage avec des propriétés telles que le caractère représentatif, la position en abscisse et en ordonnée.
  • Affichage du personnage sur la carte : La création d'une fonction display_map_and_char qui affiche à la fois la carte et le personnage.
  • Déplacement du personnage : Utilisation de la fonction input pour permettre au joueur de déplacer le personnage dans les directions haut, bas, gauche et droite.
  • Boucle de jeu infinie : Intégration d'une boucle while True pour permettre au joueur de déplacer le personnage de manière répétée.
  • Limitation des déplacements : Empêcher le personnage de sortir de la carte en modifiant la fonction de mise à jour du personnage.
  • Murs infranchissables : Rendre les murs sur la carte infranchissables pour ajouter un défi au jeu.
  • Contrôles améliorés avec curses : Utilisation de la bibliothèque curses pour des contrôles plus confortables, y compris la suppression de l'écran et le rafraîchissement.
  • Objets à collecter : Ajout d'objets représentés par des "." avec une fonction de génération aléatoire et affichage distinct.
  • Ramassage automatique des objets : Implémentation de la fonctionnalité de ramassage automatique des objets lorsque le personnage marche dessus.
  • Affichage du score : Ajout et affichage d'un score qui augmente à chaque objet collecté.
  • Génération aléatoire de cartes : Création d'une fonction generate_random_map pour générer des cartes aléatoires avec des entrées et sorties.
  • Nouveaux niveaux : Intégration d'une fonction create_new_level pour changer de niveau lorsque le personnage atteint la sortie.
  • Ajout de bombes : Possibilité pour le joueur de faire exploser les murs autour du personnage avec la touche E.
  • Fonctionnalités optionnelles : Le projet propose également des idées d'améliorations facultatives pour ceux qui souhaitent approfondir leurs compétences en programmation de jeux.

Conclusion :

Ce projet démontre les compétences en programmation de jeux en utilisant Python, avec des fonctionnalités de base et des améliorations optionnelles pour rendre le jeu plus intéressant et interactif.

Projet 2 : Extraction et d'analyse de données

Projet Java d'extraction et d'analyse de données NBA 2003 depuis basketball-reference. Scraping des classements et des joueurs avec ItemsScrapper. Analyse des performances avec ItemAnalyzer. Objectif : Analyse statistique des joueurs sélectionnés lors du repêchage NBA 2003.

Langage utilisé : java et HTML

Télécharger le projet

Description : Ce projet Java se concentre sur l'extraction de données statistiques à partir d'une page Web dédiée aux résultats du repêchage NBA de l'année 2003 (https://www.basketball-reference.com/draft/NBA_2003.html). L'objectif principal est de récupérer des informations cruciales sur les joueurs sélectionnés lors de ce repêchage pour une analyse ultérieure.

Composants principaux :

  1. ItemsScrapper :
    • Fonctionnalité : Extrayant les données brutes de la page HTML, cet composant analyse le code source et capture des informations clés telles que le classement, le nom du joueur et le nombre d'années jouées.
    • Techniques : Utilisation de méthodes indexOf et substring pour localiser et extraire les données.
  2. ItemAnalyzer :
    • Fonctionnalités : Recevant les données extraites du scrapping, cette classe offre des méthodes pour analyser les statistiques, telles que le nombre de matchs, le total d'assists, le total de points, la moyenne de points, la moyenne d'assists, la moyenne de rebonds, etc.
    • Techniques : Utilisation de méthodes simples pour calculer les statistiques demandées.

Objectifs accomplis :

  • Scrapping réussi des informations de base du repêchage NBA 2003.
  • Mise en place d'une structure de classe propre pour la manipulation des données.
  • Développement des méthodes de base pour l'analyse des statistiques.

Objectifs futurs :

  • Implémentation complète des méthodes d'analyse statistique.
  • Intégration de visualisations pour une compréhension plus intuitive des données.
  • Extension du projet pour inclure des fonctionnalités de scraping sur d'autres pages Web.

Technologies utilisées :

  • Java pour le développement du scrapping et de l'analyse.
  • Utilisation de méthodes de manipulation de chaînes pour extraire les données.
  • Potentiellement, intégration de bibliothèques pour des fonctionnalités avancées.

Contributions personnelles :

  • Conception de la structure du projet.
  • Implémentation du scrapping de base avec la classe ItemsScrapper.
  • Définition des méthodes d'analyse de base dans la classe ItemAnalyzer.

Ce projet reflète ma compétence dans la manipulation de données en Java et démontre ma capacité à concevoir des solutions pour extraire des informations utiles à partir de sources complexes, contribuant ainsi à mon expertise en développement logiciel orienté objet.
